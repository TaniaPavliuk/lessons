Типы селекторов:

1. По названию элемента

Селектор по элементу предназначен для выбора элементов по имени тега. Синтаксис: Имя тега Пример задания правила для всех элементов p на странице: /* селектор p выберет все элементы p на странице */ p { padding-bottom: 15px; }

2. По ID

В CSS коде селектор id начинается с символа # сразу после которого идет имя идентификатора. Каждый идентификатор может встречаться на странице только один раз, т. е. определенный id должен быть использован на странице только с тем тегом, для которого он предназначен.

3. По классу

CSS селектор class позволяет также как и селектор id стилизовать конкретный элемент страницы, но в отличие от id, селектор class позволяет применить свой стиль к нескольким элементам на веб-странице, а не только к одному.

4. По нескольким классам

Если нескольким селекторам нужно задать одно и то же правило, то можно написать длинно: h1 { font-weight: bold; } h2 { font-weight: bold; } h3 { font-weight: bold; } А можно перечислить все селекторы через запятую и написать всего одно CSS-правило: h1, h2, h3 { font-weight: bold; }

6. Универсальные

Универсальные селекторы CSS обозначаются обыкновенной звездочкой «*». Под звездочкой может подразумеваться любой элемент разметки HTML-страницы. "* {font-size: 16px;}" - задаст размер шрифта для текстовых элементов страницы.
